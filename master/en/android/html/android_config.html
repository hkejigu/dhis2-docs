Chapter 1. [Review ongoing]Configure DHIS2 programs to work on Android apps

To use event programs and tracker programs on the Android Event Capture and Tracker Capture apps, you must configure the programs in your DHIS2 web instance.

  1. Create a user with the correct settings in DHIS2 web instance:

    1. Create a user.

    2. Assign a user role that allow the user to access programs.

    3. Assign programs to the user role.

  2. Open the Program app and assign programs to organisation units. To work on an Android device you must have the same data capture and maintenance organisation unit as the organisation unit the program is assigned to.

    Note

    It is important to only select the actual data capture organisation unit, if you select the top organisation unit, the Android Event Capture and the Tracker Capture apps, download ALL data for the organisation unit.

  3. Open the relevant Android app and log in with the user you created.

  4. Pull down to synch with the DHIS2 web instance.

    The app downloads all programs and organisation units the user has access to.

Error message

Error message: "You don't have any programs assigned to the organisation unit"

Solution: In the Program app, assign programs to the relevant organisation units.
